Read [the announcement](https://groupincome.org/2025/11/group-income-2.5/) and check out the [music video](https://odysee.com/@groupIncome:c/Scan-Your-Soul:a).

**New Features**

- Video player! You can now upload videos to chat and play them back!
- App translated for **Korean**, **Chinese**, **Japanese**, **Spanish**, **Portuguese**, **German**, **Russian**, and **Ukrainian** (we already had French!)

**Improvements**

- Red dot now shown for any chatroom activity
- Send area handles mixed attachment types better
- Send area attachment gallery scrolls with many mixed attachment types on mobile

**Bugfixes**

- New DMs not showing up after key rotation
- Fixed handling of in-app links within chat
- Hopefully fixed delayed distribution period notification
- Fixed virtual keyboard occluding chat send message area
- Clicking on message reply links should now work for distant messages
- Chatrooms will no longer load all messages when linked to top of chatroom
